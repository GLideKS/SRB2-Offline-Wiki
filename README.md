# SRB2-Offline-Wiki

Since the long standing problem where wiki.srb2.org is so slow due to AI Scrappers (aparently), I'm just tired of it and I did my personal backup of SRB2 Wiki, holding the essential elements of the wiki for readability and style.

- You can search a specific page with this by just using a wonderful feature that Windows, Linux and probably MacOS has. **Writing up the name of the article in the folder, or using the search field.**
- **In each object's pages, some object's information are not expanded**, sorry for that. I'll get it solved when I can.
- Replaced hyperlinks via search and regular expression to redirect to the .html instead of the online webpage, allowing a mostly local navigation through the pages. (However, not all pages are downloaded)

All you need is on the SRB2 Wiki folder.

## Following tools used:
- **[SingleFile firefox extension](https://addons.mozilla.org/firefox/addon/single-file/)** to download the page when it's done to load on my computer.
- **Visual Studio Code**
